# URL
https://postscard.com/<br>
非ログイン状態の場合は記事の閲覧のみ可能です。ログインすると記事投稿やコメントが可能になります。<br>
画面上部のログインからログインページに遷移していただき、「ゲストユーザーでログインする」をクリックすると会員登録せずにログインできます。
